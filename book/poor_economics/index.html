# 貧窮的本質讀書心得

作者：阿比吉特・班納吉 (AbhijitV.Banerjee) 和埃斯特・迪弗洛 (Esther Duflo)

這本書從各個面向剖析了貧窮的原因。我閱讀起來反而不像是書。而是像一篇嚴謹且邏輯的論文。書中所關注的窮人是連飲用水和食物的獲取都有困難的。在書本最後面的總結提出了五個方面可以改善窮人的生活。第一是信息的缺乏。第二是需要肩負生活中的多種責任。第三是在市場 (保險、儲蓄等) 中取于不利地位。第四是歷史共業。第五是周圍環境人的期待。這個總結分別與其他章節的論述相呼應。但我並不是掌權者、執政者或政務官。對於平凡人或是身陷貧窮的人這些大問題知道了也無法改變。

因此我將灌注點拉回自己 (深陷貧窮的人或是常人)。關於作者提出的面向我認為只有第一、三和五是可以靠自己的決心去改變的。確實我們可能無法改變大環境。但可以改變我們能掌握的。保持學習是讀完書後我認為最重要的。這裡的學習包括技術專業、健康、理財、政策等各面向的學習。並不是要精通全部知識，而是要持續接收新知，這在未來的決斷才不會造成毀面性的結果。也才可能脫離這負面的循環。

學習到一門專業技術，才有更好的薪水待遇。學習知道健康的重要，才會去勉強自己做不喜歡但健康的事情。像是多吃蔬菜。學習知道理財的重要性，才會去找尋或學習好的投資標或保險。學習去了解政策才有可能去獲取到相關福利政策。

不斷的學習接收新知也才不會身陷周圍環境。知道世界的寬廣才不會受限於環境的眼界。看書、旅行與陌生人的交談都是為了拓展想法。先不求做到顛覆世人常識的成功，但是一旦知曉其他地方成功的案例。就不會受限於環境的期望與影響。其他人的成功即使很難複製但是作為借鑑與鼓舞自己內心還是非常有用的。書中描述窮人因為目光淺短，看不到長遠的利益。因為不知道相關訊息，錯過了社會福利。因為掌權者或其他人士的利益，扶貧計畫執行的不到位。因為生活困苦，所以轉向宗教或信念的解脫。這些問題只有自我學習才有可能掙脫這些負面的循環、枷鎖。並不是因為窮人目光短淺所以貧窮。沒有人教，周圍也沒有人說又要怎麼去意識到這個問題？我們其實與書中描述的窮人是很相近的。去健身房健身可以更健康，在未來可以省下醫療費。學習外語可以增加找到待遇更好工作的機會。但是又有多少人可以一直堅持去健身房與學外語呢？人性的弱點貪與懶不只在窮人上。我們之所以知道健康的重要去打疫苗也是因為政策規定、周圍社會的期待造就的。所以學習才有機會打破現狀。顛覆你所處的環境。脫離這些負面的循環。讀書心得的最後要分享兩段書中特別有感觸的內容。

"當你想要的一切看上去都很遙遠時，你很容易感受到動力。將目標設置的更近一些，或許是窮人實現這些目標的有效方式。" 書中這段話讓我聯想到自己的讀書計畫或時程表總是很難確實實現，甚至不了了之。眼高手低最後被失敗感打敗選擇躺平是一個很嚴重的問題。我認為自己要改善這個問題需要面對失敗，接受失敗與自身的缺點。保留這不甘的心情調整計畫與時程表較為寬鬆後更努力去執行。當你看著做不到的事情然後還是整理好心情去善後。不是那種一帆風順的天選之人，那麼面對失敗，處理失敗會是成功的必經之路。

"寡頭政治理論，腐敗的制度往往會孳生更加腐敗的制度。當政的領導者頃向于制定能夠為他們謀取財富的經濟政策。" 書中的這段讓我聯想到雅典的陶片放逐制。為了避免寡頭政治，即使你做的非常好，在保護你的名譽與財產下，還是要放逐你一段時間。寡頭政治從古至今都是一個難題。不論民主、共產、君主等制度都有其優缺點。也都會有寡頭政治的問題。烏托邦是所有人所追求的。但受限於慾望等人性的黑暗，社會的趨勢通常容易背道而馳的。而政治的變革方式甚至連書中都沒有給出一個完美的說法。我自己是認為唯有提高整體公民的素質才能將社會緩慢的往好的方向改變。
